Upload ke Roblox
Upload audio langsung ke Roblox setelah diproses. Buat API Key di create.roblox.com/dashboard/credentials (Scope: Assets Read + Write).
How to Get Your API Key
  1. Klik tombol "Create API Key" di Roblox Creator Hub
  2. Set nama untuk API key kamu (contoh: "RKDKCW ASSETS")
  3. Tambahkan permission Assets:Read dan Assets:Write (untuk deteksi moderasi)
  4. Klik tombol "Save & Generate Key"
  5. Copy API key yang muncul — kamu tidak bisa melihatnya lagi!
  6. Paste API key ke form di bawah lalu klik "Simpan"
Contoh: 9217554647
Buat API key khusus group di pengaturan group Roblox kamu (Scope: Assets Write). Berbeda dengan API key personal.
ID group dari URL: roblox.com/groups/[ID]
